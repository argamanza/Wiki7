# Wiki7 Data Pipeline - Future Page Types

Ideas for additional wiki page types to add to the data pipeline.

## Implemented

- **Season overview pages** - Main page per season linking squad, matches, transfers, stats
- **Player statistics leaderboards** - Top scorers, most appearances, assist leaders (all-time)
- **Competition pages** - Per-competition per-season fixture pages
- **Coach/Manager pages** - Full manager history with match records (via `/trainer/` spider)
- **Venue/Stadium pages** - Stadium details from Transfermarkt (via `/stadion/` spider)
- **Historical records** - Most appearances, most goals, record fees (via `/rekorde/` spider)
- **Attendance statistics** - Average, highest, lowest attendance per season (from fixtures data)
- **Honours/Trophies** - Club achievements from Transfermarkt (via `/erfolge/` spider)
- **Club transfers** - All incoming/outgoing transfers per season (via `/alletransfers/` spider)
- **Per-competition stats breakdown** - Ready via competition dropdown (Phase 4, optional)

## Wishlist (Requires Manual Curation)

- **Award pages** - Player of the Month, Season MVP, league awards (not on Transfermarkt)
- **Youth academy pages** - U19 squad data, graduates (limited TM data)
- **Derby/Rivalry pages** - Can be generated by filtering fixtures by opponent
- **Referee pages** - Not available per-club on Transfermarkt
- **Club info page** - Combine honours + records + stadium into a single page
